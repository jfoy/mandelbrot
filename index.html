The Mathematics of the Mandelbrot Set

The Mandelbrot set is one of the most famous fractals in mathematics. It's defined as the set of complex numbers c for which the function fc(z) = z² + c does not diverge when iterated from z = 0.

Each point in this visualization represents a complex number c = a + bi, where:

  • The horizontal position represents the real part (a)
  • The vertical position represents the imaginary part (b)

Starting with z₀ = 0, we compute the sequence:
z₁ = z₀² + c
z₂ = z₁² + c
z₃ = z₂² + c
...

If this sequence remains bounded (|zn| ≤ 2 for all n), then c is in the Mandelbrot set (colored black). Otherwise, the point is colored based on how quickly the sequence escapes the boundary.

Key regions to explore:

  • Main Cardioid: The heart-shaped region in the center
  • Period Bulbs: The circular regions attached to the main cardioid
  • Seahorse Valley: Around coordinates (-0.75, 0.1)
  • Elephant Valley: Around coordinates (0.3, 0)
  • Mini Mandelbrots: Smaller copies of the entire set found throughout the fractal
